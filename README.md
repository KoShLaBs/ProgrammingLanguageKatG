## 🎯 Objetivos

- Practicar creación y modificación de listas.
- Usar **indexación**, **slicing**, `append()`, `insert()`, `pop()`, `remove()`, `clear()`.
- Recorrer listas:
  - Por valor
  - Por índice
  - Con `enumerate()`
- Calcular:
  - Suma
  - Mínimo
  - Máximo
  - Promedio
- Trabajar con listas de cadenas, `len()`, `upper()`, `in`, y comprensiones de listas.
